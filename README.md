# Квадратные уравнения

Образовательный сайт для 9 класса: теория, разобранные примеры и интерактивный тренажёр.

## Разделы

- **Теория** — общий вид, неполные уравнения, дискриминант, формула корней, теорема Виета
- **Примеры** — 8 задач с пошаговым решением
- **Тренажёр** — 18 задач с проверкой ответов, подсказками и статистикой

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

## Деплой на Vercel

1. Загрузите репозиторий на GitHub
2. Импортируйте проект на [vercel.com](https://vercel.com)
3. Vercel автоматически определит Next.js и выполнит сборку

Прогресс тренажёра сохраняется в `localStorage` браузера.
