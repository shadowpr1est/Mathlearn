# Математика — теория, примеры, тренажёр

Образовательный сайт для 9 класса на **чистом HTML, CSS и JavaScript** (без React/Next.js).

Зависимостей npm **нет** — нужен только [Node.js](https://nodejs.org/) 18+.

## Разделы

- **Теория** — формулы и правила по темам
- **Примеры** — разобранные задачи с пошаговым решением
- **Тренажёр** — интерактивные задачи с проверкой, подсказками и статистикой

## Локальный запуск

```bash
node scripts/build.mjs
node scripts/serve.mjs
```

Откройте [http://localhost:3000](http://localhost:3000).

Или одной командой (если установлен npm):

```bash
npm run dev
```

- `lib/*.ts` — данные (темы, задачи, теория, примеры)
- `lib/*.js` — логика (проверка ответов, прогресс, подсказки)
- `static-src/` — стили и интерфейс

Сборка: `dist/`. Проверка: `npm test`.

## Деплой

[Vercel](https://vercel.com): `node scripts/build.mjs`, папка `dist/` (см. `vercel.json`).

Прогресс тренажёра сохраняется в `localStorage` браузера.
